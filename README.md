# [Group7] YouBike 站點投放與缺車率的空間相關性分析
本專案旨在解決共享單車因通勤時段產生的「需求不對稱」問題。  
透過空間統計學識別缺車群聚(Spatial Clustering)，並結合機器學習模型預測未來 60 分鐘的缺車風險，提供即時調度決策支援。
